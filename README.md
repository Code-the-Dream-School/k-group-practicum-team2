# Developer Community Hub

A platform for software engineering students to **learn, collaborate, and connect**.

Users can **create accounts, share resources & project ideas, bookmark content, and find teammates**.

## 📝 Description

Developer Community Hub helps students at all levels grow in tech by providing a place to share knowledge and project ideas.

### Key Features
- 👤 User authentication (Register / Login / Logout)
- 📘 Share learning resources
- 💡 Post project ideas and look for collaborators
- ⭐ Bookmark resources and projects
- 🧑‍🤝‍🧑 Community-driven collaboration
